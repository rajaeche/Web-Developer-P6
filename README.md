# HotTakes

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.2.4.



## Run Locally

To run this project
↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon server
  
  /or/
  
  node server
```
